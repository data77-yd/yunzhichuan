数据介绍

本数据集覆盖某某某区域 120 家店铺的人员流量数据，总计约 40 万条记录。数据采集周期为 2021 年 3 月至 2021 年 8 月，涵盖每日分时段客流统计。

数据字段

字段名类型说明
store_idString店铺唯一标识
dateDate日期（yyyy-MM-dd）
hourInt小时（0-23）
visitor_countInt客流量（人次）
stay_duration_avgFloat平均停留时长（分钟）
store_typeString店铺类型（餐饮/零售/服务等）

API 文档

接口地址：https://api.yunzhichuan.cn/v1/store-traffic

返回格式：JSON

请求方式：GET / POST

请求参数

参数类型必填说明
api_keyString是您的 API 密钥
store_idsArray否店铺 ID 列表，不传则返回所有
date_fromString是起始日期，格式 yyyy-MM-dd
date_toString是结束日期，格式 yyyy-MM-dd

接口备注：单次请求最多返回 10,000 条记录，超出请使用分页参数 page 和 page_size。

示例代码（PHP）

<?php $apiKey = 'your_api_key_here'; $url = 'https://api.yunzhichuan.cn/v1/store-traffic'; $params = [ 'api_key' => $apiKey, 'store_ids'=> ['ST001', 'ST002', 'ST003'], 'date_from'=> '2021-03-01', 'date_to' => '2021-08-31', 'page' => 1, 'page_size'=> 100 ]; $ch = curl_init(); curl_setopt($ch, CURLOPT_URL, $url . '?' . http_build_query($params)); curl_setopt($ch, CURLOPT_RETURNTRANSFER, true); curl_setopt($ch, CURLOPT_HTTPHEADER, [ 'Content-Type: application/json', 'Authorization: Bearer ' . $apiKey ]); $response = curl_exec($ch); $httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE); curl_close($ch); if ($httpCode === 200) { $data = json_decode($response, true); echo '获取成功，共 ' . count($data['records']) . ' 条记录'; } else { echo '请求失败，HTTP状态码：' . $httpCode; }
